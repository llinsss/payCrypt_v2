## Architecture

Tagged is built with:
- **Frontend**: React + TypeScript + Vite
- **Backend**: Node.js + Express
- **Database**: PostgreSQL
- **Cache**: Redis
- **Blockchain**: Multi-chain support (Ethereum, Base, Starknet, Core, Flow, Lisk, U2U, Stellar)
